# Contributing

Thanks for testing Seamless Texture Maker (Beta).

## Quick rules
- Keep it **client-side only** (no server required).
- Prefer **plain HTML/CSS/JS** (no build step).
- Maintain the **Skybox-style** UI (glass panel + minimal controls).
- Test in Chrome/Edge before submitting.

## How to contribute
1. Fork the repo
2. Create a feature branch
3. Commit changes with a clear message
4. Open a Pull Request

## Helpful bug reports
Please include:
- A screenshot (if relevant)
- Texture type (brick, wood, concrete, etc.)
- Your settings (strip/feather/opacity)
- Browser + OS
