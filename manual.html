How it works

Seamless Texture Maker creates a tileable texture by mirroring and softly blending edge strips: Left → Right and Top → Bottom. It keeps the result close to your original texture and avoids “creative” re-generation.

Quick start

1) Load

  • Click Choose file and select a texture (PNG/JPG/WebP).
  • The stage updates immediately with a repeating preview.

2) Fix seams

  • Adjust Strip width / Strip height.
  • Tune Feather for a softer transition.
  • Lower Opacity if the mirror looks too obvious.

3) Verify

  • Zoom out and reduce tile size to spot repetition and seams quickly.
  • Focus on straight lines and gradients (they reveal seams first).

4) Export

  • Select format (PNG/WebP/JPG).
  • Click Export to download the seamless tile.

Controls

Output size

Source (fit square) crops the center of the image to a square and uses that resolution. Fixed sizes (512–4096) generate a consistent output resolution.

Tip: test at 1024, export final at 2048/4096 if needed.

Horizontal tiling

  • Strip width: how many pixels are copied from the left edge.
  • Feather: softness of the blend mask.
  • Opacity: strength of the overlay (lower = less mirror visible).

Vertical tiling

  • Strip height: how many pixels are copied from the top edge.
  • Feather and Opacity: same meaning as horizontal.

Reset

Resets the working tile back to the original cropped base and re-applies the current settings.

Preview navigation

  • Drag = pan
  • Mouse wheel = zoom
  • Shift + wheel = tile size (more/less repeats)
  • Double click = reset view

Best way to detect seams: set tile size to 64–128, zoom out slightly, and scan for lines.

Export

PNG

Best quality, biggest files. Good for archive and further editing.

WebP

Excellent quality/size balance. Recommended for web projects.

JPG

Smallest, but adds compression artifacts (may reveal seams in flat gradients).

Naming

Export uses a clear filename pattern: seamless_edgeblend_1024x1024.png (example).

Best practices

Good starting settings

  • Strip: 40–90 px
  • Feather: 25–60
  • Opacity: 70–100%

Structured textures (tiles, bricks)

  • Keep strips smaller (40–80)
  • Opacity ~60–90%
  • Avoid very high feather (can blur grout lines)

Organic textures (concrete, dirt)

  • Use larger strips (60–140)
  • Feather 45–75
  • Opacity 85–100%

If seams remain

  1. Increase Feather slightly
  2. Increase Strip size slightly
  3. Reduce Opacity a bit (often helps)
  4. Re-check at tile size 64–128

FAQ

Why do I still see seams on bricks/tiles?

Structured materials require alignment (straight lines must match). This method blends edges cleanly, but it can’t “rebuild” broken geometry. Use smaller strip sizes and lower opacity to keep lines crisp.

The mirrored edge looks obvious. How do I reduce it?

Reduce Opacity (try 70–85%) and slightly increase Feather. Also reduce the strip width/height.

Is this tool offline?

Yes. No AI, no server calls. Everything runs locally in your browser.

Which browser should I use?

Chrome or Edge are recommended for the most consistent Canvas behavior. Firefox is usually fine as well.

Why does “Source (fit square)” crop the texture?

Seamless tiles are typically square and repeat in both directions. Cropping keeps the workflow simple and predictable.

Any quick “power user” tricks?

Use small tile size (Shift + wheel) + zoom out to instantly see repetition. If the seam is a hard line, raise Feather; if it’s a “ghost double image”, lower Opacity.
